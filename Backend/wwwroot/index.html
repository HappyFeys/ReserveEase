API Documentation - AuthController

This documentation provides information about the authentication API endpoints.

Endpoint: POST /auth/login

Description: Authenticates a user and creates a session.

Request Body

                                                                    {
"username": "string",
"password": "string"}

Responses

200 OK - Login successful

{ “message”: “Login successful” }

400 Bad Request - Already logged in

{ “message”: “Already logged in” }

401 Unauthorized - Invalid credentials

{ “message”: “Invalid credentials” }

Endpoint: POST /auth/logout

Description: Logs out the authenticated user and ends the session.

Responses

200 OK - Logout successful

{“message”: “Logout successful” }

400 Bad Request - Not logged in

{ “message”: “Not logged in” }

Connexion WebSocket


const socket = new WebSocket('ws://localhost:5001/ws');

Description: Cette ligne crée une nouvelle connexion WebSocket à l’URL spécifiée (ws://localhost:5001/ws).

Événement onopen


socket.onopen = function (event) {
console.log('Connexion établie');
};

Description: Cet événement est déclenché lorsque la connexion WebSocket est établie avec succès. La fonction associée affiche un message dans la console indiquant que la connexion est établie.

Événement onmessage


socket.onmessage = function (event) {
alert('Message reçu du serveur : ' + event.data);
};

Description: Cet événement est déclenché lorsqu’un message est reçu du serveur. La fonction associée affiche une alerte avec le contenu du message reçu.

Événement onerror


socket.onerror = function (error) {
console.log('Erreur WebSocket :', error);
console.log("Détails de l'erreur WebSocket : ", error.message);
};

Description: Cet événement est déclenché lorsqu’une erreur se produit dans la connexion WebSocket. La fonction associée affiche les détails de l’erreur dans la console.

Événement onclose


socket.onclose = function (event) {
if (event.wasClean) {
console.log(`Connexion fermée proprement, code=${event.code} raison=${event.reason}`);
} else {
console.log('Connexion interrompue');
}
};

Description: Cet événement est déclenché lorsque la connexion WebSocket est fermée. La fonction associée vérifie si la fermeture a été propre et affiche un message approprié dans la console.

Envoi de message


document.getElementById('sendMessageButton').addEventListener('click', function () {
socket.send('Message du client');
});

Description: Cette ligne ajoute un écouteur d’événements au bouton avec l’ID sendMessageButton. Lorsque le bouton est cliqué, un message est envoyé au serveur via la connexion WebSocket.
